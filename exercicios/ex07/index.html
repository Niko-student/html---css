Principais formatações de texto

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (Não semantica).

Nesta frase, tenho um termo em destaque usando a tag <strong> (Semantica).

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag <i> (Não semantica).

Nessa frase temos um termo em ênfase usando a tag <em> (Semantica).

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>.

Também podemos trocar as cores dos marcadores usando o CSS.

Texto Maior e Menor

Podemos criar textos menores e maiores usando as tags <small> e <big>.

Texto Deletado

Podemos criar textos deletados usando a tag <del>.

Texto Inserido

Também podemos criar textos inseridos usando a tag <ins>.

Também podemos criar textos sublinhados (Não semantica)usando a tag <u>.
